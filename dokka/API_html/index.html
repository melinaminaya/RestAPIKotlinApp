API de Integração

API para uso das interfaces de usuário para interação com o serviço de comunicação Autotrac. Esta API implementa métodos que possibilitam a interação de uma ou mais interfaces de usuário com o serviço de comunicação Autotrac que deverá estar previamente instalado e ativado no dispositivo móvel.
